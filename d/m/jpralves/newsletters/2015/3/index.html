Open Source ^

Inkscape - A powerful, free design tool

Estando na sua versão 0.91, lançada em Janeiro de 2015, trata-se de uma ferramenta que permite fazer desenhos vectoriais.

O Inkscape é um software de edição profissional de gráficos vectoriais que corre em Linux, Windows e Mac OS X. É usado tanto por profissionais como por entusiastas para criar um tipo variado de gráficos como ilustrações, icons, logotipos, diagramas, e gráficos para a Internet. O Inkscape usa o standard SVG (Scalable Vector Graphics) como o seu formato nativo.

Este software tem um conjunto de ferramentas comparáveis ao Adobe Illustrator ou CorelDraw. Permite exportar e importar de vários formatos de ficheiros incluindo SVG, AI, EPS, PDF, PS e PNG.

Baseado numa interface simples o Inkscape tem um conjunto de funcionalidades muito alargado das quais se destacam as seguintes:

  • Criação de objectos
  • Manipulação de Objectos
  • Preenchimento e contornos
  • Operações sobre linhas
  • Suporte para Texto
  • Representação
  • Outras

Circuitos ^

Aqui é apresentado um circuito simples que poderá ser construído com componentes.

Regulador de Tensão 5V

Estes circuitos foram feitos recorrendo a quatro reguladores de tensão muito comuns - o LM7805, o LM2940, o LM1117 e o AMS1117.

O LM7805 é um regulador linear de voltagem que é muito popular, pois para o colocar a funcionar não são necessários mais componentes. É um componente muito barato. Devido às suas características, a forma como faz a redução da voltagem de saída é feito à custa de dissipação de calor o que o torna pouco eficiente. Para que o LM7805 funcione efectivamente é necessário que tenha uma voltagem de entrada miníma de 7.3V. Este suporta uma corrente máxima de 1A. Alguns modelos suportam até 1.5A. Os condensadores são aconselhados e em alguns caso necessários para diminuir/eliminar os efeitos das frequências introduzidas pelos outros elementos do circuito. Servem igualmente para que sejam atenuados os efeitos dos picos de consumo.

O LM2940 é de uma geração diferente, mantendo no entanto a compatibilidade de pinos com o LM7805. Trata-se de um Low-dropout (LDO) Linear Regulator com melhor eficiencia que o LM7805 sendo que no entanto precisará obrigatoriamente de condensadores.

O LM1117 tem um pinout diferente dos dois anteriores.

O AMS1117 além de ser de um formato diferente mantém a compatibilidade com o LM1117 em termos dos pinos. É um integrado mais evoluido que os anteriores e com uma eficiência superior.

IC Formato Input Max Amp DataSheets
LM7805 TO-220 7V a 20V 1A Link
LM2940-5.0 TO-220 6V a 26V 1A Link
LM1117 TO-220 6.5V a 15V 800mA Link
AMS1117-5.0 SOT-223 6.5V a 12V 800mA Link

Artigo do Maker ^

Projeto interessante publicado por um maker.

polargraph

Este projeto apresentado pelo Fred Kaems no seu site e no instructables mostra como construir uma máquina que permite desenhar na vertical. Este artigo foi originalmente publicado neste link pelo seu autor.

Video

O projecto foi documentado pelo autor à medida que o ia construindo. Tudo começou com uma Todo-List que se subdividia em duas partes. A componente mecânica e a componente do software.

O maker usou os seguintes componentes para a construção do projecto:

  • 1 x Arduino Uno
  • 1 x Shield de motores
  • 2 x Nema-17 stepper motors. Estes são idênticos aos usados na impressora prusa i3. Usam 1.3A e têm 1.8 graus por passo. (200 passos para uma volta completa)
  • 1 x Motor Servo para levantar / baixar a caneta.
  • 1 x Fonte para os stepper motors.
  • Fios para os servo.
  • Algumas peças impressas / cortadas a laser.

Os stepper motors foram montados no topo da frame.

Foi necessário para o desenho existir um contra-peso que foi feito com recurso a uma garrafa de agua.

O autor fez uso de um software disponibilizado neste link. Este software foi desenvolvido pela Marginally Clever que tem um kit com o Makelangelo . O software teve que ser adaptado para a configuração que o Fred Kaems montou.

Para o suporte da caneta foram feitas umas peças para montar a Gondola com uma impressora de corte a Laser.

Ferramentas ^

As Ferramentas que usamos no dia-a-dia explicadas ao detalhe.

Chaves de parafusos

Chaves manuais para apertar e desapertar parafusos. Não têm grande ciência mas existem diversos formatos além dos dois tipos básicos que conhecemos. A chamada chave de fendas e a chave de estrela ou "Phillips".

Existem com a ponta fixa ou com pontas (bits) intermutáveis.

Quer por questões práticas ou por questões relacionadas com dificultar a desmontagem dos equipamentos os fabricantes desenvolveram um conjunto adicional e diverso de chaves. Não sendo uma lista exaustiva os formatos mais comuns que existem são:

  • Fendas (a)
  • Estrela (b)
  • Pozidriv (c)
  • Torx (d)
  • Allen - Sextavada (e)
  • Robertson ou quadrada (f)
  • Tri-Wing (g)
  • Torq-Set (h)
  • Spannner (i)

Importa ainda mencionar as chaves que sendo de um destes tipos são de utilização especial.

Os Busca pólos são umas chaves que têm a ponta de fendas que através de uma lâmpada de néon associada a uma resistência de valor muito alto permite verificar numa instalação de corrente alterna (AC) qual é o pólo neutro e qual é a fase. Na fase a lâmpada é iluminada.

Dica:

Caso tenham um parafuso cuja rosca da cabeça tenha ficado moída existem alguns métodos que podem ser usados para tentar remover o parafuso com sucesso. Existe um método que consiste em usar uma Dremel para fazer um buraco fundo para usar uma chave de fendas. Outro método passa por usar um elástico grosso entre o parafuso e a chave para se tentar desaparafusar.

Por fim existe um tipo de chaves especialmente desenhadas para desaparafusar parafusos com a rosca moida - tipicamente funcionam bem para parafusos Estrela, Fendas, Torx e Robertson.
